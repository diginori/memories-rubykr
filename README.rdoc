== Setting

* Attention: 시드(Seed) 데이터에 다수의 사진이 포함되어 있기 때문에, Cloning과 Seeding에 시간이 소요됩니다.

* git clone git://github.com/sungheeekang/memories-rubykr.git
* cd memories-rubykr
* bundle
* rake db:migrate
* rake db:seed (오래 걸립니다)
